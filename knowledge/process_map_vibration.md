# Process Map — Bearing Vibration Increase

## Triggers
- Overall vibration above alert/alarm
- Step change after transient (sync, load swing, trip/restart)
- Narrowband 1x or 2x rise

## Quick checks
1. Compare DE / NDE / gearbox probes for pattern
2. Check phase and orbit if available (not always in CSV)
3. Lube oil supply pressure & temperature
4. Process coupling: load, speed, exhaust temp concurrent changes
5. Recent maintenance: alignment, coupling work, balance shots

## Severity cues
- Slow climb over weeks → fouling, foundation soft foot, thermal bow residual
- Step + process event → mechanical insult or sensor jump
- Multiple bearings simultaneous → structural / foundation / measurement reference

## Actions
- Do not ignore OEM alarm levels
- Capture high-speed vibration if system supports
- Plan inspection scope: coupling, bearings, rotor path, foundation bolts
